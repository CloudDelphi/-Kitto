To use the KEmployee demo you need to:

1) Setup Firebird 2.5 with a database alias "EMPLOYEE" defined in aliases.conf like:
   EMPLOYEE = C:\Program Files (X86)\Firebird\Firebird_2_5\examples\empbuild\EMPLOYEE.FDB
2) Optionally edit database connection parameters in Home\Metadata\Config.yaml

URL to launch demo
------------------
http://localhost/kitto/kemployee
