To use the DbDemos demo you need to:

1) Locate position of your DbDemos.gdb database (by default it is located under C:\Users\Public\Documents\Embarcadero\Studio\19.0\Samples\Data\dbdemos.gdb)
2) Make sure your Interbase Server is up and runnig
3) Then edit database connection parameters in Home\Metadata\Config.yaml
4) Launch Home\DbDemos.exe with -a parameter if you want to use the GUI
5) Login with user "guest" and password "password" (look into FileAuthenticator.txt to see other users)
